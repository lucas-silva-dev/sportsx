- Instale as dependências com _yarn_ ou _npm_:

```sh
$ yarn install
$ npm install
```

- Inicialize o Front

```js
$ yarn start
 //ou //
$ npm start
```

Se tudo ocorrer bem, o projeto vai abrir por padrão em:

```sh
http://localhost:3000/
```

Caso a porta esteja ocupada, vai buscar a próxima porta disponível a partir da 3001, 3002, 3003...
